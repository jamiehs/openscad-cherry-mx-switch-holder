# Cherry MX Keyswitch Holder

Designed so a series of Cherry MX style switches can be pushed in with a friction fit.

```
// Wall Thickness
wall_thickness = 1;

// Hole Diameter
hole_diameter = 5.2;
hole_radius = hole_diameter/2;

// Switch Base Size
switch_base_size = 14;

// Switch Count
switch_count = 4;

// Switch Hole Depth
switch_hole_depth = 9.5;

// Base Height
base_height = 12;

// Skirt Height
skirt_height = 2;

// Skirt Offset
skirt_offset = 6;

// Skirt Radius
skirt_radius = 2;

// Switch Spacing
switch_spacing = 6;
```
